# board2
미니 게시판 개인 프로젝트입니다.



# 사용한 기술스택
- JAVA 11
- Spring Boot
- Spring Data Jpa
- Querydsl
- Thymeleaf
- h2


# 구현 기능
게시글 CRUD 
페이징
게시글 제목 검색 필터
